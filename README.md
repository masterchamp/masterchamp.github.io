2. Allgemeines

Diese App dient der Verwaltung einer persönlichen Karnivoren-Sammlung. Der Schutz Ihrer Daten ist uns wichtig. Die App verarbeitet personenbezogene Daten ausschließlich im notwendigen Umfang.

3. Erhobene Daten

Die App speichert ausschließlich Daten, die vom Nutzer selbst eingegeben werden, darunter:

Pflanzeninformationen

Bestandsdaten

Ereignisse und Notizen

Kauf- und Verkaufsinformationen

Fotos der Pflanzen

angelegte Bezugsquellen (Shops)

4. Speicherung der Daten

Alle Daten werden ausschließlich lokal auf dem Gerät des Nutzers gespeichert.
Es erfolgt keine automatische Übertragung an externe Server oder Dritte.

5. Berechtigungen

Die App kann folgende Berechtigungen verwenden:

Kamera – zum Aufnehmen von Pflanzenfotos

Fotos/Medien/Speicher – zum Auswählen und Speichern von Bildern

Internet – ausschließlich zum Prüfen von App-Updates oder Öffnen externer Webseiten (z. B. Shoplinks)

6. Weitergabe von Daten

Es werden keine Daten an Dritte weitergegeben, verkauft oder analysiert.

7. Backups

Der Nutzer kann freiwillig Backups erstellen. Diese werden lokal als Datei gespeichert und nur vom Nutzer selbst verwaltet.

8. Nutzerrechte

Da alle Daten lokal gespeichert werden, kann der Nutzer jederzeit:
